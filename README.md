Refrence source - https://en.wikipedia.org/wiki/Line_drawing_algorithm
Results pixel is set to Texture2D displayed on a flat Quad GameObject

## How to Run
Open the project in Unity `2022.3.21f1` and press Play in the main menu scene:
- `Assets/LineDrawingAlgorithm/Examples/MainMenu/MainMenu.unity`

From the main menu, pick an algorithm and then click 2 points on the quad to draw a line.
